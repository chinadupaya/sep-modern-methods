# sep-modern-methods

## Prerequisites
- node/npm installed in machine

## Getting Started (backend)
install dependencies of our frontend
```bash
cd sep-backend
npm install
```

to run our tests:

```bash
npm test
```

... and run the development server:

```bash
npm start
```
## Getting Started (frontend)
install dependencies of our frontend
```bash
cd sep-frontend
npm install
```

run the development server:

```bash
npm run dev
```

Head to localhost:4200 on your web browser to access the application!

**note**
- both frontend and backend servers should be running for the program to work
